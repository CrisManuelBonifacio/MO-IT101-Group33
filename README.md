MotorPH Payroll System (Milestone 2)

This project is a Java-based payroll system developed using NetBeans.

Features:
- Login system
- Employee information management
- Weekly hours worked calculation
- Gross weekly salary computation
- Net weekly salary computation
- Government deductions (SSS, PhilHealth, Pag-IBIG)
- Late deduction included in gross salary
- Bulk payroll processing
